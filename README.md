# CLI Commands Reference

## What Is This Project?

This is a command line reference guide. It covers the most commonly used commands across three major shells — **Bash**, **Zsh**, and **PowerShell** — with practical examples and clear explanations.

The goal is simple: help you understand what each command does, how it works, and when to use it.

## Why Does This Exist?

If you have ever opened a terminal and felt lost, this project is for you. The command line can seem intimidating at first, but once you understand the basics, it becomes one of the most powerful tools on your computer.

This guide breaks down shell commands into easy-to-follow chapters. Each chapter focuses on a specific topic, explains the commands involved, and shows real examples so you can try things out yourself.

## What You Will Learn

- How to navigate your filesystem using the terminal
- How to create, move, rename, and delete files and directories
- How to use wildcards, flags, and shortcuts to work faster
- How the shell interprets your commands behind the scenes
- Tips and habits that make everyday tasks smoother

## How To Use This

1. Clone this repository
2. Open the folder for the shell you want to learn
3. Start with Chapter 1 and work your way through
4. Try the examples in your own terminal as you go

Each command follows the same format — a short introduction, the syntax, a working example, and an explanation of what happened. No assumptions, no jargon, just clarity.

## Found A Mistake?

No guide is perfect. If something is wrong, unclear, or does not work as described, you can pinpoint it by opening an issue on this repository. Mention:

- What the mistake is
- Where it is located (chapter name and section)
- What the correct behavior should be

Your feedback makes this project better for everyone who uses it.
